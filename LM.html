Linear Models (LM)

Let $y_1, y_2, ..., y_n$ be the responses from $n$ subjects, each response follows independently a normal distribution: \[ y_i \sim N(\mu_i, \sigma^2), i=1, 2, \cdots, n \]

A linear model is to relate the mean $\mu_i$ with some characterics of each subject, $x_{i1}, x_{i2}, \cdots, x_{ik}$, called predictors, covariates, independent variables, etc. These covariates are considered to have potential effects on the mean of the response: \[ \mu_i=\beta_0 + \beta_1 x_{i1}+ \beta_2 x_{i2} + \cdots + \beta_k x_{ik}\] or \[ y_i=\beta_0 + \beta_1 x_{i1}+ \beta_2 x_{i2} + \cdots + \beta_k x_{ik} + \epsilon_i=x_i \beta+\epsilon_i, \epsilon_i \sim N(0, \sigma^2), i.i.d. \] where $x_i= (1, x_{i1}, ..., x_{ik})$ is the observation of the covariates from subject $i$, the parameter vector $\beta = (\beta_0, \beta_1, ..., \beta_k)^T$ is the regression coefficients, and $\epsilon_i$ is the random variation in y that is not explained by the covariates in the model. The random variation may be due to other variables that affect y but are not known or not observed, or simply measurement errors. The parameters $\beta_0, \beta_1, ..., \beta_k $ are called regression coefficients.

Let $ \mathbf{y} = (y_1,y_2, \cdots, y_n)^T $ be the observation vector. Then the linear model can be rewritten as \[ \mathbf{y}=\mathbf{X \beta} + \mathbf{\epsilon}, \mathbf{\epsilon} \sim N(0, \sigma^2 \mathbf{I_n}) \] where $\mathbf{X}=(x_1^', x_2^', ..., x_n^')' $ is the observed covariate matrix.

The simple regression, multiple regression, polynomial regression, ANOVA models, ANCOVA models are all linear models. The main purpose of fitting a linear model to the observed data can be

  • to investigate the relationship of the response variable and the covariates, and the importance of the covariates on the response
  • to predict the response for some different values of the covariates
  • to find an optimum response
No matter what purpose is, the first step is to estimate the parameters $\mathbf{\beta}$ and the common standand deviation $\sigma$.

Some notes on linear models:

  • By “linear” it means “linear in parameter”. It does not mean “linear in covariate”. So polynomial regression is a special case.
  • Normal distributed means the response variable is a continuous variable.
  • Some transformation of the response variable may be needed for being normally distributed.
  • Covariates can be continuous variables, factors with several levels (such as treatment groups), or even a constant 1 for the intercept.
  • Factors should be coded before performing analyses. There are different ways in the coding systems. Check Coding Systems for Categorical Variables in Regression Analysis (ucla.edu). Many software packages do the coding but care should be taken when interpretation of the regression parameters. Different coding will result in different interpretations. In SAS, the categorical variables are declared by the CLASS statement with several options for coding.
  • It is a good practice to perform some standardization of the covariates.
  • Check whether there are collinearity between the covariates
